
# AI-Powered Smart Crane Monitoring & Predictive Maintenance System

## Overview

The AI-Powered Smart Crane Monitoring System is a real-time industrial dashboard designed to monitor crane performance, predict maintenance requirements, and improve operational efficiency.

The application simulates an Industrial IoT environment using live sensor data and interactive charts.

## Features

* Real-time crane monitoring
* Load monitoring
* Wear & tear analysis
* Fuel tank monitoring
* Working hours tracking
* Temperature monitoring
* Motor RPM monitoring
* AI-based maintenance recommendations
* Automated alert system
* Interactive Chart.js graphs
* Responsive design
* Live updates every 20 seconds

## Dashboard Metrics

* Overall health score
* Current load capacity
* Fuel level
* Working time
* Temperature
* Vibration status
* Motor RPM
* Daily usage analytics
* Last 90 days performance

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/AI-Smart-Crane.git
```

2. Open the project folder:

```bash
cd AI-Smart-Crane
```

3. Open `index.html` in your browser.

## How It Works

* Sensor values are simulated in JavaScript.
* Dashboard values update automatically every 20 seconds.
* Graphs update in real time using Chart.js.
* AI logic generates maintenance recommendations.
* Alerts are triggered when sensor values exceed safe limits.
